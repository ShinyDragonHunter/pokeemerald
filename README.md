# Optimised Pokémon Emerald

This will eventually be an optimised Pokémon Emerald with redundant and/or unused animations, code and text strings
removed to provide a cleaner and more organised workspace for people whilst trying to keep the game, story and link features unchanged.

# Goals:

* Duplicate and unused code, sprite animations and text strings will be completely removed

* Code will eventually be functionally equivalent but smaller to vanilla code

* The story and link capability features will remain untouched to keep compatibility with vanilla games

Anyone is welcome to help out and contribute to the project.

The following ROM is built:
